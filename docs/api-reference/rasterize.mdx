`POST /v1/rasterize`

Convert each page of a PDF to a PNG image. Powered by PDFium (the PDF engine used in Chrome).

```
Authorization: Bearer forme_sk_...
```

---

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `pdf` | `string` | Yes | Base64-encoded PDF bytes |
| `dpi` | `number` | No | Resolution in dots per inch. Range: 72-300. Default: `150`. |

---

## Examples

<CodeGroup>

```bash curl
curl -X POST https://api.formepdf.com/v1/rasterize \
  -H "Authorization: Bearer forme_sk_abc123..." \
  -H "Content-Type: application/json" \
  -d "{
    \"pdf\": \"$(base64 -w0 invoice.pdf)\",
    \"dpi\": 200
  }"
```

```javascript Node.js
import fs from 'fs';

const pdf = fs.readFileSync('invoice.pdf').toString('base64');

const res = await fetch('https://api.formepdf.com/v1/rasterize', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer forme_sk_abc123...',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ pdf, dpi: 200 }),
});

const { pages } = await res.json();

// Save each page as a PNG
pages.forEach((page, i) => {
  fs.writeFileSync(`page-${i + 1}.png`, Buffer.from(page, 'base64'));
});
```

```python Python
import requests
import base64

with open("invoice.pdf", "rb") as f:
    pdf_b64 = base64.b64encode(f.read()).decode()

res = requests.post(
    "https://api.formepdf.com/v1/rasterize",
    headers={
        "Authorization": "Bearer forme_sk_abc123...",
        "Content-Type": "application/json",
    },
    json={"pdf": pdf_b64, "dpi": 200},
)

data = res.json()
for i, page in enumerate(data["pages"]):
    with open(f"page-{i + 1}.png", "wb") as f:
        f.write(base64.b64decode(page))
```

</CodeGroup>

## Response

`200 OK` with JSON:

```json
{
  "pages": [
    "<base64-encoded PNG for page 1>",
    "<base64-encoded PNG for page 2>"
  ]
}
```

Each element in the `pages` array is a base64-encoded PNG image.

---

## DPI Guide

| DPI | Use Case | Approximate Size (Letter) |
|-----|----------|--------------------------|
| 72 | Quick thumbnails | ~100 KB/page |
| 150 | Standard preview (default) | ~300 KB/page |
| 300 | Print quality | ~1 MB/page |

Higher DPI produces larger images and takes longer to process.

---

## Use Cases

- **Preview images**: Generate thumbnails for document listings
- **Visual verification**: Check redaction results before distributing
- **Image-based workflows**: Convert PDFs for systems that only accept images
- **Comparison**: Visually diff two versions of a document
